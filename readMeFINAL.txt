HOW TO USE:
1. Open the program
2. Read the catalogue of functions
	2a. Decide what function you want to execute
	2b. To choose a function, enter the desired function's corresponding number.

TIPS AND TRICKS:
If you want to search a list, you must sort it first.
To do this, create a list. Once you've done that, USE THE SORT FUNCTION!
Now, Search Away!!!
